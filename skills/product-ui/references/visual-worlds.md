When asking Q4b, suggest 2–3 worlds that fit the domain. Don't list all 11 raw.

| Domain | Suggest |
|---|---|

| Finance / Analytics | Clean/Clinical, Premium/Refined, Bold/Graphic |
| Developer Tools | Industrial/Utilitarian, Retro-Futuristic, Clean/Clinical |
| SaaS / Internal Tools | Clean/Clinical, Bold/Graphic, Premium/Refined |
| Healthcare | Clean/Clinical, Soft/Pastel |
| Food / IoT / Hardware | Bold/Graphic, Rustic/Textured, Retro-Futuristic |
| Consumer / Lifestyle | Illustrated/Playful, Soft/Pastel, Bold/Graphic |
| E-commerce / Catalog | Clean/Clinical, Editorial/Magazine, Premium/Refined |
| Education / Learning | Illustrated/Playful, Clean/Clinical, Soft/Pastel |
| Marketing / Content | Bold/Graphic, Editorial/Magazine, Illustrated/Playful |
| Travel / Transportation | Editorial/Magazine, Clean/Clinical, Premium/Refined |

**Full list (offer if user wants more options):**
- Illustrated / Playful — warm, organic, characters
- Rustic / Textured — kitchen, workshop, natural materials
- Premium / Refined — dark bg, gold accents, precision
- Clean / Clinical — white space, single accent, minimal
- Bold / Graphic — high contrast, strong typography
- Editorial / Magazine — dramatic type, asymmetry, layers
- Retro-Futuristic — sci-fi, neon on dark, geometric precision
- Brutalist / Raw — exposed structure, high contrast, no decoration
- Art Deco / Geometric — ornamental, symmetry, metallic accents
- Soft / Pastel — gentle, rounded, approachable
- Industrial / Utilitarian — functional, dense, mechanical
- Let you decide

## Visual World → Design Language

The visual world is an influence, not a costume. It should be felt in spacing, rhythm, typography weight, and motion — not just applied as a color palette swap.

Use as the design compass — commit to it completely or don't use it at all.

| Visual World | Typography | Color Approach | Visual Language |
|---|---|---|---|
| **Illustrated / Playful** | Rounded display + friendly sans | Warm, saturated, generous contrast | Organic shapes, illustrated icons, character elements |
| **Rustic / Textured** | Serif display + body sans | Warm browns, cream, muted amber | Subtle texture, warmth over precision, imperfect edges |
| **Premium / Refined** | Thin serif + light sans | Dark bg, gold/warm accent, restrained | Hairlines, geometric precision, scientific diagram feel |
| **Clean / Clinical** | Geometric sans | White bg, single strong accent | Grid, breathing room, no decoration |
| **Bold / Graphic** | Heavy display | High contrast, single dominant color | Large type as graphic element, bold blocks |
| **Editorial / Magazine** | Dramatic display + refined body | Strong contrast, unexpected accent | Asymmetry, overlap, type as image, diagonal flow |
| **Retro-Futuristic** | Mono or geometric display | Neon on near-black, glows | Grid lines, scanlines, terminal energy, tight spacing |
| **Brutalist / Raw** | Heavy grotesque or system font | Near-black + raw accent | Exposed structure, visible borders, zero decoration |
| **Art Deco / Geometric** | Tall serif + geometric sans | Black/gold or navy/cream | Symmetry, ornamental borders, metallic touches |
| **Soft / Pastel** | Rounded sans | Muted pastels, low saturation | Soft shadows, rounded corners, gentle gradients |
| **Industrial / Utilitarian** | Condensed grotesque or mono | Dark gray, desaturated accent | Dense layout, grid lines, functional over decorative |

## Font Pairings by Visual World

Never use as display/heading fonts: Inter, Roboto, Arial, Helvetica, Space Grotesk, system-ui.

| Visual World | Pairing |
|---|---|
| Premium / Refined | Cormorant Garamond + DM Sans |
| Bold / Graphic | Bebas Neue + IBM Plex Mono |
| Editorial / Magazine | Playfair Display + Source Serif 4 |
| Retro-Futuristic | Share Tech Mono + Rajdhani |
| Clean / Clinical | DM Sans + DM Sans (weight contrast only) — or Outfit, Plus Jakarta Sans |
| Illustrated / Playful | Nunito + Nunito Sans |
| Rustic / Textured | Lora + Source Sans 3 |
| Art Deco / Geometric | Josefin Sans + EB Garamond |
| Brutalist / Raw | IBM Plex Mono + IBM Plex Sans (or system font deliberately) |
| Soft / Pastel | Quicksand + Nunito Sans |
| Industrial / Utilitarian | Barlow Condensed + Barlow |

Line length: 65–75 characters per line for body text. Line height: 1.5–1.75 for body.
