# Phase 4 Feedback Questions

Pick the set that matches what was built. Ask as plain text, not inside a code block.

---

## Multi-step flows (booking, checkout, wizard, onboarding)
1. Did all steps work end-to-end — could you complete the full flow?
2. Does each step show only what's needed for that step, or is it cluttered?
3. Is it clear how to go back, and does state carry forward correctly?

## Dashboards / analytics / monitoring
1. Can you find the most important number/status within 3 seconds?
2. Is anything missing that you'd need to make a decision?
3. Does the density feel right — too much, too little, or just right?

## Forms / data entry
1. Is every field clearly labeled with the right format hints?
2. Is the validation feedback helpful — does it tell you what's wrong, not just that something is wrong?
3. Is the primary submit action obviously the next step?

## Catalogs / search / listing pages
1. Can you find what you're looking for without knowing what it's called?
2. Do the filters surface the dimensions that actually matter for this domain?
3. Does the no-results state give you a useful next step?

## Default (everything else)
1. Does the visual feel match what you had in mind?
2. Is anything missing that a real user would need to accomplish their goal?
3. Should anything look, feel, or behave differently?
